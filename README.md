# Snippets (recetas autocontenidas)

Este repositorio contiene **ejemplos mínimos reproducibles** por tema y lenguaje.

## Estructura
- `python/`
- `java/`
- `web/`
- `devops/`
- `iot/`

## Regla de oro
Cada snippet debe incluir:
1) `README.md` (qué hace + cómo ejecutar)
2) Código mínimo (lo estrictamente necesario)
3) (Opcional) salida esperada o captura

## Cómo aportar
- Si surge un error en un live (Twitch), primero documéntalo en:
  https://github.com/willedtech/knowledge-base/discussions
- Luego conviértelo en snippet con un PR aquí.
